**Nodejs MySql Admin**

MySql database viewer/editor written in nodejs. It's in development phase.

![Picture1](http://i.hizliresim.com/pXVOG0.png)


To use it change password in *mysql.js* to your MySql password, run:

	npm install

And then

	gulp
